GitHub repository ထဲတွင် Dockerfile နှင့် start.sh ကိုတင်ပါ။
Railway တွင် New Project ဖန်တီးပြီး ထို Repo ကိုချိတ်ပါ။
Variables (Settings) ထဲတွင် အောက်ပါတို့ကို ထည့်ပေးရပါမည်-
NGROK_AUTH_TOKEN: (သင့် Ngrok dashboard မှ token)
VNC_PASSWORD: (Login ဝင်ချင်သော စကားဝှက်၊ ဥပမာ - MySecretPass)
Deploy လုပ်ပြီးပါက Railway logs (Deploy logs) ထဲတွင် Ngrok URL (ဥပမာ tcp://0.tcp.ngrok.io:12345) ကိုရှာပါ။
သင့်ကွန်ပျူတာရှိ VNC Viewer (RealVNC သို့မဟုတ် TightVNC) တွင် ထိုလိပ်စာကိုထည့်ပြီး ချိတ်ဆက်နိုင်ပါပြီ။
